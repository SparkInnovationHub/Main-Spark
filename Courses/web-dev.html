Your Learning Journey

Week 1 - Onboarding and Fundamentals of the Web

The first week introduces the course structure and expectations, provides an overview of the internet and essential web technologies, and guides students in setting up their development environments and tools.

Week 2 - Front-End Basics

In the second week, students learn the basics of HTML to create web pages, get an introduction to CSS for styling, and understand the Document Object Model (DOM).

Week 3 & 4 - Advanced CSS

Weeks three and four dive deeper into CSS properties and techniques, cover responsive design to ensure web pages look good on all devices, and introduce CSS frameworks like Bootstrap.

Week 5 - JavaScript Basics

The fifth week focuses on the JavaScript programming language, teaching variables, data types, basic syntax, and how to manipulate the DOM with JavaScript.

Week 6 - JavaScript Deep Dive

In the sixth week, students explore advanced JavaScript concepts such as closures and callbacks, understand asynchronous JavaScript with Promises and async/await, and learn to use JavaScript libraries like jQuery.

Week 7 & 8 - Fundamentals of Back-End Development

Weeks seven and eight introduce server-side programming, teach about databases and how to interact with them, and guide students in building basic APIs using Node.js and Express.js.

Week 9 - Bonus: Introduction to UI/UX

The ninth week covers the principles of user interface (UI) and user experience (UX) design, introduces wireframing and prototyping tools, and discusses best practices for designing intuitive and user-friendly interfaces.

Week 10 - Authentication and Deployment of Websites

In the tenth week, students learn to implement user authentication and authorization, explore different methods of deploying websites, and get introduced to deployment platforms like Heroku and Netlify.

Week 11 & 12 - Capstone Project Completion and Presentation

Weeks eleven and twelve are dedicated to applying the skills learned throughout the course to build a complete web application, working on individual or group projects, and preparing and delivering a final presentation of the capstone project.

Week 13 - Graduation

The final week involves reviewing the course content, celebrating achievements, discussing next steps and career opportunities in web development, and receiving certificates of completion and final feedback.
